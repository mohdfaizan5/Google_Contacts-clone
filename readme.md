### Usage:

#### 1. Clone repo
```bash
git clone https://github.com/mohdfaizan5/Google_Contacts-clone
```

#### 2. Rename .env_example -> .env
```env
PORT = 3000
MONGO_URL = "<mongo atlas url> or <mongo local host>"
```

#### 3. Install dependencies
```bash
npm install
```

#### 4. Run 
```bash
npm run dev
```
